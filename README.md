# Infinit Butchery Module

Butcheries Industry Module for the Maxnate Infinit Platform.

## Features

- Multi-tenant meat retail, processing, and distribution
- Carcass tracking and yield management
- Cold chain monitoring
- Weight-based pricing
- Online ordering and delivery
- Pluggable payment integrations
- 10+ website templates

## Installation

```bash
bench get-app infinit_butchery
bench --site yoursite install-app infinit_butchery
```
